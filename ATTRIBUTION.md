# Attribution (CC BY 4.0)

The non-code content in this repository is licensed under **CC BY 4.0**. If you
share, publish, redistribute, or adapt that content, you must provide
attribution.

## Recommended attribution format

Include all of the following (where reasonable):

- **Title**: "BioSimulant Models" (or the specific model/space name)
- **Author**: "BioSimulant contributors"
- **Source**: a link to the repository hosting the content
- **License**: "CC BY 4.0" and a link to the license
- **Changes**: a brief note if you modified the content

Example:

> BioSimulant Models (BioSimulant contributors). Source: <repo link>. Licensed
> CC BY 4.0. Changes: modified parameters and wiring.

## What counts as "content" here

Typically includes (non-exhaustive):

- `model.yaml`, `space.yaml`, wiring/config YAML/TOML
- documentation under `docs/` and `README.md`
- any curated metadata describing models/spaces

## What counts as "code" here

Typically includes (non-exhaustive):

- Python/TypeScript/JS source under `scripts/`, `libs/`, `templates/`, and model
  `src/` folders
